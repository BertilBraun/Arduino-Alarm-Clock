# Arduino Alarm Clock
 A Alarm clock based on the Arduino NodeMcu 

## Utility

 Server side Settings available to be changed at update.php
 - Alarm Timing
 - Snooze Timing
 - Dark Mode
 - Dissabeling of Screen
 
 Timing based on Server side Time, Requested from Arduino via HTTP Request

## Finished Circuit

 - PN2222 Transistor
 - 1N4001 diode
 - Pin zu Motor -> 270Ω
 - Pin zu Button -> 10kΩ
